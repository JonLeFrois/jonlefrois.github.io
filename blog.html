Microsoft Azure vs. Amazon AWS

-based on my personal experience-

  This blog comparing Azure and AWS is based on unequal experiences with both of these cloud services. During my internship with Bestpass, I gained experience with Azure as that is what the company used to host their application. With AWS my experience comes from my current knowledge gained from working through an AWS certification course. That being said, my comparisons will be more from hands on experience and less about the difference in technical specifications. For more information on the technical specs of these two cloud services, check out the GeeksForGeeks article linked in the "Learn More!" section.

Azure

  • Second largest cloud service provider
  • Easily works with Microsoft SQL Server
  • Strong security and authentication procedures that are well documented on their implementation
  • Smooth compatibility with other Microsoft tools
  • Easier setup and a structure that makes more sense

AWS

  • Largest cloud service provider
  • Tons of customization options, can be tricky for beginners
  • There is a lot of documentation which can get confusing quickly
  • Seems to offer a wider range of services
  • Gives you more control over your cloud service
